Privacy Policy — Clan Panel PTT Bridge (Chrome Extension)

Дата: 2026-01-31

1. Какие данные мы собираем
    Расширение не собирает, не хранит и не передаёт персональные данные пользователя, историю браузера, содержимое страниц, пароли, платежные данные или точную геолокацию.
2. Как работает расширение
    Расширение передаёт события push‑to‑talk между веб‑страницей и локальным агентом на компьютере пользователя. Для этого оно подключается по WebSocket только к локальному адресу ws://127.0.0.1:8765 (и/или ws://localhost ).
3. Передача третьим лицам
    Данные не передаются третьим лицам. Расширение не отправляет данные на внешние серверы.
4. Хранение данных
    Расширение не сохраняет пользовательские данные.
5. Безопасность
    Соединение осуществляется только с локальным агентом на устройстве пользователя (loopback). Удалённый код не загружается и не выполняется.
